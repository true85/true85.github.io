## SSMS With, 임시테이블
### with
```sql
WITH 변수명1 AS(
    만들 퀴리문
)

SELECT c1, c2
FROM 변수명1
```

### 임시테이블
```SQL
CREATE TABLE #TBL이름(
    조건
)
```
임시테이블로 복사
```SQL
SELECT * INTO #TBL이름 FROM 원본TBL [조건문]
```
